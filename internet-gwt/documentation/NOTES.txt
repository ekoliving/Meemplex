

Resources
---------

Types:
- raster images: png, jpeg,
- vector graphics: svg

Labels
- a label for each generic resource (e.g. blinds-icon1)

Themes
- themes consist of pointers to resources from generic resource labels


Resource Location
-----------------

- map a label to a path (including theme)
- revert to default if not found

The label "icon-blinds1" might map to path "/resource/default/icon-blinds1.svg"
